网络小游戏 《五子棋Pro》

网络游戏 后端集群 综合性作品 精致小游戏 网友喜爱的作品
      这是一款经过3次重塑设计的小游戏作品，是一款极具综合能力表现的作品。该作品以经典民间游戏“五子棋”为创作主题，全面实现了常用的游戏功能。
      本游戏的官方地址为： https://game.domoe.cn/gobang3 ，同时支持微信小游戏平台以及字节跳动小游戏平台，全平台用户注册数量为80万(截止2021.3)，平台小程序码为：

本作品相比于其他同类作品的优点有（功能上）：

      1.支持人机、单机双人、支持编号棋盘、支持禁手规则；
      2.网络对弈模式功能稳定，支持断线重连；
      3.积分规则合理；
      4.好友对弈，支持输入房间号、支持小程序卡片、支持面对面扫码，方便快捷；
      5.局内交流、礼物系统完善，互动体验极佳；
      6.完备的商城、任务系统，增添游戏粘性。
      7.支持对局复盘，二次推演。

技术亮点：

      1.本作品使用自研组件 “NaiveNet” 开发设计，仅使用 Cocos Ctreator 游戏设计引擎，后端均未使用非本人设计的第三方游戏设计插件或引擎（帧同步服务等均未使用）。
      2.游戏内所有算法均为本人自行编写，包括：AI算法、禁手算法等；
      3.合理的地方使用缓存技术提高系统效率，任务系统、排行榜系统等；
      4.后端分布式集群架构，非单点服务器，可根据用户活跃数量不停机弹性增减配，支持单点故障自动容灾；
      5.权限控制极佳，本游戏通常只存在AI代下作弊，其他作弊方式普遍不可行（自研游戏后端对弈权限控制严格，未使用帧同步技术）；

开源插件 NaiveNet

NIO - 高性能 分布式 轻量级 Netty 技术栈 TCP/IP 协议栈
      NaiveNet是一款刘昕禹自研的网络通信组件。
      官方地址： http://www.naivenet.cn
      开发文档： 访问文档 强烈推荐略读本文档以更清晰了解该组件功能
      Github仓库： 访问仓库
      NaiveNet它是基于NIO网络模型设计完成的网络通信组件，这套组件可以让我们的HTML5、各类小程序/小游戏以长连接（Web Socket）通信模式开发应用，面向需求的高度封装使得开发者无需关注长连接的网络通信细节从而高效的开发应用。如果你想做一个实时交互应用比如一个实时的小游戏，或者是在线的多平台聊天室，甚至你有更好的实时交互创意，那么学会使用NaiveNet会让你事半功倍。

技术亮点：

      1.使用Netty技术栈，NIO网络通信模式；
      2.基于全双工创意的Request请求模式，十分适用于Web开发者；
      3.内置授权、SESSION机制；
      4.自带断线恢复机制；
      5.天然的分布式构架；
      6.跨平台、跨语言开发。

微教务系统 - 沈阳工业大学

PHP、Vue 数据库 Oracle + MySQL 经典信息系统 全校师生的最爱 精致美观大气
      这是刘昕禹本科毕业时的离别之作，倾力设计，但自己却没能用到。
      微信公众号： 沈阳工业大学教务通    微信小程序： 沈阳工业大学教务通

      该作品是19年时期沈阳工业大学的教务系统中的颜值担当，系统部署于学校机房现已常年运营。该作品结合微信公众平台以及企业微信公众平台的开放能力，对接学校教务系统数据库提供的移动端快捷查询功能，最为亮点的是该作品除了提供常见的学科、绩点、四六级成绩查询外，提供了电子成绩单、电子在读证明、电子年级排名（PDF格式）的创建与企业核验功能，该功能在2020年疫情期间发挥巨大作用，电子档案功能是辽宁首家高校上线，后续又被多家高校模仿。成绩单界面精美，排版合理，不错版，作品质量极佳。此外，该作品还提供师生最爱的“可视化课程表”，该课程表颜值高，适应各种复杂排课情况，自带时间线，广受好评。

作品亮点：

      1.系统设计经验丰富，常年运行从未故障、宕机；
      2.系统资源回收算法合理，档案过期自动回收；
      3.无论是电子档案，还是课程表，UI设计十分精美，体现扎实的技术功底；
